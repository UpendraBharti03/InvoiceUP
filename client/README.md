# InvoiceUP client

### 100% [TypeScript](https://www.typescriptlang.org/).

### Technologies used
- React JS
- Vite
- TypeScript
- Axios
- Tanstack react router

### Develop

To develop this repo, run the following command:

```
yarn dev
```

## FAQs
1. How to install a dependency?

   Use Yarn workspace, ex: install axios
```
yarn add axios
```
